# Kakulator 
Project ini merupakan pengembangan kalkulator scientific berbasis web yang dirancang untuk melakukan perhitungan aritmatika dan trigonometri secara cepat dan akurat. Aplikasi ini dibangun menggunakan HTML, CSS, dan JavaScript, serta dilengkapi fitur live result yang menampilkan hasil perhitungan secara otomatis tanpa harus menekan tombol sama dengan (=), sehingga meningkatkan efisiensi dan kenyamanan pengguna. Disini kalian juga bisa mengakses link di bawah ini .

 https://aditya-tres.github.io/kakulator-aditya-akbar-baru/

## Features
- Aritmatika Dasar (Penjumplahan, Pengurangan, Perkalian, Pembagian , Persen) 
- Trigonometri (Sin, Cos, Tan)
- History Perhitungan
- Panel Trigonometri Interaktif
- Tombol kontrol (C untuk menghapus seluruh input, DEL untuk menghapus satu karakter terahir, Tanda () kurung untuk operasi matamatika bertingkat, tombol = untuk menampilkan hasil perhitungan)
